丢帧导致卡顿

fps代表GPU渲染画面的频率，Hz代表显示器刷新屏幕的频率。
我们谈到掉帧，是指GPU渲染画面频率降低。
比如跌落到30fps甚至20fps，但因为视觉暂留原理，我们看到的画面仍然是运动和连贯的。
